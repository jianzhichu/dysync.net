# dysync.net

#### Description
dysync.net 是一款基于 .NET Core 6.0 和 Vue 开发的小软件，主要用于同步抖音收藏夹的视频，解决抖音收藏视频经常失效的问题，让你喜欢的视频能真正存储到自己的硬盘上。
